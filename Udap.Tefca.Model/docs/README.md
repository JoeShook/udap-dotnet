# Udap.Tefca.Model

TEFCA profile models and extensions for the [UDAP .NET SDK](https://github.com/JoeShook/udap-dotnet).

This package provides TEFCA-specific authorization extension types (e.g., `tefca-ias`) and DI registration for use with the core UDAP server packages.

## Usage

```csharp
services.AddUdapTefcaExtensions();
```
